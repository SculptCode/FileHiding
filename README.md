
# FileHidding

This simple piece of code helps you hide files in the file location you enter from windows using "arttib" commands.

```
Usage: Shift / CTRL
```


## Setup

Download required libraries

```cmd
  pip install keyboard
```

Run the project

```cmd
  python fileHiding
```
